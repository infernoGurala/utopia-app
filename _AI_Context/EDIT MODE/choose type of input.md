
there are 3 main types of selections 

1. [[Upload a file]]
2. [[add a question&answer in utopia]]
3. [[some text block in utopia]]
4. [[special blocks]]

- every section block starts with it own heading, the highest heading of all "#" in mermaid. for file it is "Files", for Q/A it is "Questions", for text the user have to decide the title and description also.

---
special input for 
1. Bold
2. italic
3. link [title](link)
4. highlight
5. strike through
6. undo, redo options
7. single latex

- all the special inputs are above the keyboard to choose from.
---

- any block can be deleted, and it can be also brought back with undo option. 
- but the undo is not possible when the file was pushed to github.
