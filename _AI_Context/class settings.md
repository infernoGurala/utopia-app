this feature is a option inside every class. allows people to who has write access to edit the features respective to the selected class.
- Class settings only visible to only the people who have writer access.

Features include
--------------------------------------
1. Edit writers - Add/remove the writers from the class. 
2. Share the class link. 
3. Class name is permanent/cannot be changed.
4. Delete class option -- Critical DANGER warning procedure to this must be followed.
5. writers can be added via selecting the people in the same class, a view must be opened to select the persons to make a writer.
6. a timetable option to edit the timetable for the current class. this edit logic is to be followed as existing timetable logic.

Special Case
-----------------------------
- FOR COMMUNITY NOTES/UNIVERSITY NOTES, THERE IS A EXCEPTION TO BE MADE. 
- every one in the community notes will have the few writer mode options.
- all the people can edit any file/folder.
