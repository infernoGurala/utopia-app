# UTOPIA v3.0 — Feature List

## Core Features
- Class and university friends feature with student count display.
- Separate streaks for friends and university groups in the friends panel.
- Folder and file creation across repository with icon and color customization.
- Editor mode for ease of access (markdown support, learn tab).
- Course structure: 5 units per course, subtitle shown in grey.
- "About this app" section explaining functionality and creator info.
- Multi-campus support (different campuses workable).
- Remove password gate if possible.
- Display previous semester CGPA and SGPA scores.
