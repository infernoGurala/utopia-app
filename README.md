# Utopia - Academia life simplified

> *"Simplifed, to only what matters."*

A student academia life app built with Flutter,Firebase and supabase.

## Features
- Flutter app with 3-tab navigation (Focus, Academia, Profile)
- Services (Auth, Firestore, Groq AI, Supabase, and Cloudinary)
- Attendance (one touch, all data gathered and simplified)
- student notes management (personal, and community)
